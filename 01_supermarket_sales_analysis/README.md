# Supermarket Sales Analysis

## Overview
This project analyses supermarket transaction data from three branches.
The objective is to understand sales performance, customer behaviour,
and key factors influencing revenue.

## Key Questions
- Which branches generate the most revenue?
- Do members spend more than normal customers?
- Which product lines perform best?
- How does customer satisfaction vary?

## Dataset
- Source: Public practice dataset
- Rows: 1,000
- Columns: 11
- Currency: Assumed USD (not specified in data)

## Tools Used
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Files
- `Supermarket_Sales_Analysis.ipynb` — Main analysis notebook
- `data/supermarket_sales.xlsx` — Dataset
